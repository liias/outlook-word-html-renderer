Pptdocumentsettings Element (Microsoft PowerPoint)

Defines document settings for a presentation when it is saved to the Web.

Attribute Description
browsersupport Contains one of the following string constants specifying the version of Microsoft Internet Explorer supported: dual, v3, or v4. Dual specifies both versions 3 and 4.
donotresizegraphics Determines whether the graphics are resized to fit the Web page. The default value is False.
framecolors Contains one of the following string constants specifying the frame colors.
BlackTextOnWhite Black text on white
BrowserColor Default colors used by browser
PresentationSchemeAccentColor Presentation colors (accent color)
PresentationSchemeTextColor Presentation colors (text color)
WhiteTextOnBlack White text on black (default)
hidenavigation Determines whether the slide navigation controls are hidden. The default value is False.
hideslideanimation Determines whether the slide animation is hidden. The default value is False.

Contained In

Presentation

Remarks

The Web Options dialog box is used to modify the document settings.

Example

This example shows a document setting of white text on black for the frame colors.

<p:pptdocumentsettings framecolors="WhiteTextOnBlack" />
