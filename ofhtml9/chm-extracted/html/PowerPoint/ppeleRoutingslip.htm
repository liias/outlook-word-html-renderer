Routingslip Element (Microsoft PowerPoint)

Defines the address information for a routed presentation.

Attribute Description
offset The offset into the information. The default value is -1. This attribute is specified if the size is greater than 0.
size The size of the routing slip in bytes. The default size is 0.

Contained In

Presentation

Remarks

Routing slips are used to route presentations to one or more recipients. The routing slip stores the recipient information in order.

Example

This example shows a routing slip.


<p:routingslip size="130" offset="0"/>
