Printoptions Element (Microsoft PowerPoint)

Specifies the print options that can be saved for a presentation.

Attribute Description
colormode Contains one of the following string constants specifying the color mode for printing.
bAndW Black and white with grays
fullColor Color (default)
pureBAndW Pure black and white (no grays)
frameslides Defines an integer frame value for each slide.
printhidden Determines whether hidden text is printed. The default value is False.
printwhat Contains one of the following string constants specifying what is printed.
buildSlides Built slides
handouts2 Handouts, two to a page
handouts3 Handouts, three to a page
handouts6 Handouts, six to a page
notes Speaker notes
outline Slide show as an outline
slides Slides (default)
scaletofitpaper Determines whether slides are scaled to fit the paper. The default value is False.

Contained In

Presentation
