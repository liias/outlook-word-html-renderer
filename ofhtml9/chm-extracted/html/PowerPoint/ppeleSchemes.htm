Schemes Element (Microsoft PowerPoint)

Defines a collection of color schemes in a presentation.

Contained In

Master

Subelements

Colorscheme

Remarks

Each scheme is defined by the Colorscheme element. At least one Colorscheme subelement must be specified.

Example

Two different color schemes are saved.



<p:schemes>
<p:colorscheme colors="#FFFFFF,#000000,#808080,#000000,#00CC99,#3333CC,#CCCCFF,#B2B2B2"/>
<p:colorscheme colors="#FFFFFF,#000000,#333333,#000000,#DDDDDD,#808080,#4D4D4D,#EAEAEA"/>
</p:schemes>
