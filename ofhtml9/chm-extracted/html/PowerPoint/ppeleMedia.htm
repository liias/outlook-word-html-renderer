Media Element (Microsoft PowerPoint)

Describes the audio/video portion of a slide.

Attribute Description
endtrack Ending track in a track sequence if the media is a compact disk.
loops Defines whether the media will loop around. If True, the media will loop. The default value is False.
href Defines the path pointing to the file that contains the audio/video media. Required only for media objects that reference an external file.
isnarration Defines whether the media is a narration. If True, the media is a narration. The default value is False.
length Specifies the length of the media sequence in milliseconds. The default value is 0.
rewinds Defines whether the media will rewind when it comes to the end of the sequence. If True, the media will rewind. The default value is False.
starttrack Beginning track if the media is a compact disk.
type

Defines the audio/video format. One of the following values is required for this attribute.

avi Files that are encoded in the Microsoft Audio-Video Interleaved format.
cdAudio Audio tracks on a compact disk.
embeddedWAV Embedded files that are encoded in the Microsoft Windows Audio-Video Interleaved format.
linkedWAV Linked files that are encoded in the Microsoft Windows Audio-Video Interleaved format
mci Audio/video information that provides commands for the Microsoft Windows Media Control Interface.
midi Files that conform to the Musical Instruments Digital Interface specification.
quicktime Files that are encoded in the Apple QuickTime format.

Contained In

Slide

Subelements

Sound

Remarks

All subelements are optional.

Use the OnMouseClick element to play a media object. For example: 


<p:onmouseclick action="playMedia"/>

Example

A media object is created that is an embedded WAV file 5 seconds long. The actual sound derives from a Sound subelement that references a WAV. 


<p:media type="embeddedWAV" length=5000>
<p:sound id="040754" name="LAURE.WAV" href="sound001.wav"/>
</p:media>
