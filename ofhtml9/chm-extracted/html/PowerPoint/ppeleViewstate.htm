Viewstate Element (Microsoft PowerPoint)

Specifies the view state of a presentation.

Attribute Description
hideoutlineicons Determines whether the outline icons are hidden. The default value is False.
horizbarstate Defines the state of the horizontal bars. The default value is 1.
manualadjustleft Determines whether the left value can be manually adjusted. The default value is False.
manualadjusttop Determines whether the top value can be manually adjusted. The default value is False.
nosnaptogrid Determines whether the view snaps to a grid. The default value is False.
notestextformatting Determines whether the notes text is formatted. The default value is False.
notestextscale Defines the scale of notes text. The default value is 100.
outlineformatting Determines whether the outline view is formatted. The default value is False.
outlinescale Defines the scale of the outline. The default value is 33.
preferviewset Determines whether the view preferences have been set. The default value is False.
restoreleft Defines the default left value of a slide. The default value is 327.
restoretop Defines the default top value of a slide. The default value is 909.
scale Defines the scale of the view.
slidehref Defines the path to the slide.
snaptoshape Determines whether the view snaps to a shape. The default value is False.
sortdraft Specifies whether the draft view is sorted. The default value is False.
sortscale Defines the scale for sorting. The default value is 66.
type Contains one of the following string constants specifying the type of view.
handout Handout
notes Notes
notesMaster Notes master
notesText Text of notes
outline Outline
outlineMaster Outline master
slideMaster Slide master view
slideShow Slide show view
slideShowFullScreen Slide show full screen
slideSorter Slide sorter view
slideView Slide view
titleMaster Title master
visualBasic Visual Basic view
vertbarstate Defines the state of the vertical bars. The default value is 1.
