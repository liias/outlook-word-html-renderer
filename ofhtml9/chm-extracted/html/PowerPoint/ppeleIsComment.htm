IsComment Element (Microsoft PowerPoint)

Determines whether a text box in PowerPoint will be a comment or not. Comments are tracked so that they can be turned on or off as a group.

Contained In

Rect (VML)

Remarks

When saved to HTML, a master VML Rect element will be created with the IsComment element embedded within it. For example:



<v:rect id="_x0000_s2052"
style='position:absolute;left:1.25pt;top:1.25pt;width:208.75pt;height:56.125pt;' fillcolor="#fcff91">
<v:shadow on="t" obscured="t" offset="6pt,6pt"/>
<v:textbox style='mso-fit-shape-to-text:t'/>
<p:iscomment/>
</v:rect>

Then a shape is created using the Rect ID value with the actual data from the comment in it. For example:



<div v:shape="_x0000_s2052" class=O>
<div style='mso-line-spacing:"100 50 0";position:absolute;top:1.2%;left:1.26%;
height:3.61%'>
<span style= 'font-family:Arial;font-size:67%;color:black'>
<b>Veronica:</b>
</span>
<span style= 'font-family:Arial;font-size:67%;color:black;display:none'>&#13;</span></div>
<div style='mso-line-spacing:"100 50 0";position:absolute;top:6.5%;left:1.26%;
height:3.61%'>
<span style='font-family:Arial;font-size:67%;color:black'>
This is my comment.
</span>
</div>
</div>

Note that the name of the commenter ("Veronica") is set off in bold, while the comment ("This is my comment.") is not.
