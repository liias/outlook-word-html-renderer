Slideshowsettings Element (Microsoft PowerPoint)

Specifies the settings used for a slide show.

Attribute Description
browse Determines whether the slide show is in browse mode. The default value is False.
endslide Index of the last slide in the presentation range. The default value is the last slide in the presentation.
hidescrollbar Determines whether the scroll bars are hidden. The default value is False.
kiosk Determines whether the slide show is in kiosk mode. The default value is False.
loop Determines whether the slide show loops continuously until the ESC key is pressed. The default value is False.
manualadvance Determines whether the transition between slides is advanced manually by mouse clicks. The default value is False.
namedshow Specifies the name of a custom slide show.
pencolor Color name or RGB value of the pen used to draw on slides during a presentation. The default color is defined by the color scheme for text.
restarttime Time between slide shows when running in kiosk mode. The value is measured in 1/1000 of a second. By default, the presentation does not restart.
skipbuilds Determines whether slides in a custom show are skipped. The default value is False.
skipnarration Determines whether the slide show narration is skipped. The default value is False.
startslide Index of the first slide in the presentation range. The default value is 1.

Contained In

Presentation

Remarks

The slide show settings are generated by selecting options in the Set Up Show dialog box.

Example

This example shows the settings saved for a slide show that will loop in a window continuously with a restart time of 5 minutes and a pen color defined as the default color for text.


<p:slideshowsettings loop="t" browse="t" restarttime="300000" pencolor="schemeText">
</p:slideshowsettings>
