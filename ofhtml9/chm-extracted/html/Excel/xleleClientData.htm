ClientData Element (Microsoft Excel)

This element contains a the properties and data of an ActiveX control, Office Art graphic, or Microsoft Office Web component.

Attribute Description
ObjectType Specifies the type of component.

Contained In

Body (HTML) or any other element that surrounds content.

Subelements

WebChart
