PlotArea Element (Microsoft Excel)

Contains the formatting properties of the plot area of a chart.

Contained In

Chart

Subelements

Axis, Border, DataTable, Font, Graph, Height, Interior, Left, Top, Width

Remarks

All subelements are optional. If this element is not specified, Microsoft Excel or the Chart component automatically determines the formatting for the plot area.
