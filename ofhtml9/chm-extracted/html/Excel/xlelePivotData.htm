PivotData Element (Microsoft Excel)

Contains a description of the state of PivotTable data, including whether members are expanded and the position of a view.

Contained In

PivotTable

Subelements

Column, InvertedColumnMember, InvertedRowMember, LeafColumnMember, LeafRowMember, Left, Top

Remarks

All subelements are optional. You can specify more than one Column, InvertedColumnMember, InvertedRowMember, LeafColumnMember, and LeafRowMember subelement.
