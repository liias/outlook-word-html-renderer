PivotItem Element (Microsoft Excel)

Contains the properties of an item in a PivotTable field or a filter item of a PivotList component.

Contained In

PivotField, PLPivotField

Subelements

For a PivotList component: Hidden, Member, Name, SourceName

For a PivotTable field: CachePosition, DrilledMember, Formula, FormulaIndex, Hidden, HideDetail, Missing, Name, SourceName

Remarks

The Name subelement is required, but all other subelements are optional.

For a PivotList component, either the Member or SourceName element can be specified, but not both.
