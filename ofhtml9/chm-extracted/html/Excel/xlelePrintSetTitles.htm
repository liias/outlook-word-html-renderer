PrintSetTitles Element (Microsoft Excel)

Specifies whether print titles for a worksheet are set based on a PivotTable report or worksheet.

Contained In

PivotTable

Remarks

If <x:PrintSetTitles/> is specified, row print titles are set to the rows that contain the PivotTable report’s column field items, and column print titles are set to the columns that contain its row items. If this element is not specified, the print titles for the worksheet are used.
