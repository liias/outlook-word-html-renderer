SplitType Element (Microsoft Excel)

Specifies how points in a data series are placed on a primary pie chart and its bar or secondary pie chart.

Contained In

Graph

Remarks

This element contains one of the following string constants: Automatic, Custom, Percent, Position, or Value.

If the Automatic constant is specified, the Chart component or Microsoft Excel determines the placement of points. If the Custom constant is specified, the SplitValue element contains a user-defined value specifying how the points are placed. For all other constants, the SplitPos element specifies how the points are placed.
