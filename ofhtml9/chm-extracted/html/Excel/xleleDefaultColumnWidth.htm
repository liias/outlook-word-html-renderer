DefaultColumnWidth Element (Microsoft Excel)

Contains the default column width of a worksheet.

Contained In

WorksheetOptions

Remarks

One unit of column width is equal to the width of one character in Microsoft Excel's Normal cell style. For proportional fonts, the width of the character 0 (zero) is used. If this element is not specified, the default width is 8 units.
