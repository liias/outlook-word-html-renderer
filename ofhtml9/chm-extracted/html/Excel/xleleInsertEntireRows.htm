InsertEntireRows Element (Microsoft Excel)

Specifies whether entire rows are inserted to accomodate new data when a query table is refreshed.

Contained In

QueryTable

Remarks

If <x:InsertEntireRows/> is specified, entire rows are inserted to accomodate new data and unused cells are cleared. If this element is not specified but <x:OverwriteCells/> is specified, then existing cells are used for new data and unused cells are cleared. If neither element is specified, new cells are inserted to accomodate new data and unused cells are deleted. Both elements cannot be specified at the same time for a single query table.
