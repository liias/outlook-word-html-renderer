Horizontal Element (Microsoft Excel)

Specifies the horizontal justification for text.

Contained In

Alignment

Remarks

This element contains one of the following string constants specifying the justification: Automatic, Center, Distributed, Justify, Left, or Right.

If Automatic is specified, Microsoft Excel automatically determines the horizontal justification. Distributed can be specified only in East Asian language versions of Excel.
