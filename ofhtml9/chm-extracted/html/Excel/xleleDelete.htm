Delete Element (Microsoft Excel)

Specifies whether an axis is drawn, a series is deleted, or a data label or legend entry is removed.

Contained In

Axis, DataLabels, LabelEntry, LegendEntry

Remarks

If <x:Delete/> is specified, the axis is not drawn, the series is deleted, or the data label or legend entry is removed.

For an axis, the properties of other features, such as gridlines and crossing information, are kept intact. If this element is specified for an axis in Microsoft Excel, the axis is not selectable even though gridlines and crossing information is specified in subelements of the Axis element. If this element is specified in the Chart component, the axis will not have color, labels, or tick marks.
