GroupType Element (Microsoft Excel)

Specifies how groups are ordered in a PivotTable field.

Contained In

PivotField

Remarks

This element contains one of the following string constants specifying how the numeric or date values are ordered: Range, Seconds, Minutes, Hours, Days, Months, Quarters, or Years. If this element is not specified, the values are ordered by Range.

For a PivotList component, the GroupType element is used for automatic date fields and contains one of the following string constants specifying how the date values are ordered: Days, Months, Quarters, or Years. When the value of this element is Months and the value contained in the PLGroupType element is Weeks, the field is the "Weeks" field of a weekly automatic date field set.
