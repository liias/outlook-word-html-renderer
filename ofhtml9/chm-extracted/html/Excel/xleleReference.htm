Reference Element (Microsoft Excel)

Contains the properties of a consolidation reference when in a SourceConsolidation element, or a cell range when in the ConsolidationReference element.

Contained In

ConsolidationReference, SourceConsolidation

Subelements

For the SourceConsolidation element: ConsolidationReference, Item

Remarks

For the SourceConsolidation element, the ConsolidationReference subelement is required and the Item subelement can be specified up to four times.
