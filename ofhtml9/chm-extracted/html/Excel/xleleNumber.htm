Number Element (Microsoft Excel)

Contains the index number of either an HTML table in a Web page or of a pane in a worksheet window, the number formatting style of a chart object, or a numeric operand.

Contained In

Axis, Crn, DataLabels, HTMLTables, LabelEntry, Moper, Pane, ShowBubbleSize, ShowPercent, ShowValue

Subelements

For a chart: BuiltInFormat, FormatString, SourceLinked

Remarks

For chart number formatting, either the BuiltInFormat or FormatString subelement must be specified, but all other subelements are optional.

When the data comes from a PivotList component, the Number subelement of the Pane element is required and contains the integer 3.
