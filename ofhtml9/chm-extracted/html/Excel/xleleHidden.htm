Hidden Element (Microsoft Excel)

Specifies whether the name of a range is visible in the Define Name dialog box in Microsoft Excel, or whether an item is hidden or filtered from a PivotTable report.

Contained In

ExcelName, PivotItem

Remarks

If <x:Hidden/> is specified, the name or item is hidden or filtered.

Examples

See the examples in the ExcelName and PivotTable element topics.
