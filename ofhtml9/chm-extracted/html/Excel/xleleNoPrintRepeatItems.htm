NoPrintRepeatItems Element (Microsoft Excel)

Specifies whether row, column, and item labels appear in the first row of each page when a PivotTable report is printed.

Contained In

PivotTable

Remarks

If <x:NoPrintRepeatItems/> is specified, labels are printed only on the first page. If this element is not specified, the row, column, and item labels appear in the first row of each printed page.
