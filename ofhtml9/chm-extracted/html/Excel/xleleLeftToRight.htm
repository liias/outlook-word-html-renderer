LeftToRight Element (Microsoft Excel)

Specifies whether a sort is done left to right by column or top to bottom by row, or whether a workbook or worksheet is printed left to right.

Contained In

Print, Sorting

Remarks

For a sort, if <x:LeftToRight/> is specified, the data is sorted by column, but if the element is not specified, the data is sorted by row.

For a workbook or worksheet, if <x:LeftToRight/> is specified, the data is printed left to right, but if the element is not specified, the data is printed right to left.
