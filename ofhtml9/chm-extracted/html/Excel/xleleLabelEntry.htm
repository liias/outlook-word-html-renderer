LabelEntry Element (Microsoft Excel)

Specifies the formatting for the data label of a trendline and allows for custom placement of data labels on a chart.

Contained In

DataLabels, DisplayUnits, Trendline

Subelements

Alignment, Border, Caption, DX, DY, Font, Index, Interior, LabelSeparator, LeaderLines, Number, Position, ShowBubbleSize, ShowCategoryName, ShowLeaderLines, ShowLegendKey, ShowPercent, ShowSeriesName, ShowValue

Remarks

The Index subelement is required, but all other subelements are optional.

The ShowCategoryName, ShowLegendKey, ShowSeriesName, and ShowValue subelements can be specified for all types of series.
