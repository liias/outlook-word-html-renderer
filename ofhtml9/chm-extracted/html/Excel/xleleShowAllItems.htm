ShowAllItems Element (Microsoft Excel)

Specifies whether empty members are displayed even if they do not have aggregates.

Contained In

PivotField, PLPivotField

Remarks

If <x:ShowAllItems/> is specified, the empty members are visible.

When the data is saved from a PivotList component and the data source is not an OLAP provider, ShowAllItems is specified for all but the first field on the axis. The DisplayEmptyMembers subelement of the PivotAxis element specifies whether the items are visible. When the Web page is opened in Microsoft Excel and the axis has no fields, the content of the DisplayEmptyMembers element is used to specify whether empty members are displayed.
