Line Element (Microsoft Excel)

Contains the formatting properties of a line.

Contained In

Axis, DataPoint, DropLines, ErrorBars, Graph, HiLowLines, LeaderLines, Series, Trendline

Subelements

Color, ColorIndex, LineStyle, Weight

Remarks

All subelements are optional.

If this element is not specified, Microsoft Excel or the Chart component determines the default line formatting and style.
