IMEMode Element (Microsoft Excel)

Specifies the status of the Input Method Editor (IME).

Contained In

DataValidation

Remarks

This element can contain one of the following values:

Value Description
0 No control
1 On
2 Off (English)
3 Disabled
4 Hiragana
5 Katakana
6 Half-width Katakana
7 Full-width alphanumeric
8 Half-width alphanumeric
9 Full-width Hangul
10 Half-width Hangul

This element is used only when East Asian language support is installed and enabled.
