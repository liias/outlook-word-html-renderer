Value2 Element (Microsoft Excel)

Contains a text string specifying the value or formula used in the second part of conditional text formatting.

Contained In

Condition

Remarks

When you specify a condition based on the value of a cell and you specify the Between or NotBetween qualifiers, use the Value1 element to specify one boundary of the comparison range and the Value2 element to specify the other.

When you specify a formula, you must start the formula with an equal sign (=).
