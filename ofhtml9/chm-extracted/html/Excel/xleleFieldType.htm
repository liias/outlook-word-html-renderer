FieldType Element (Microsoft Excel)

Specifies the cell data type formatting for a column.

Contained In

FormatSettings

Remarks

This element contains one of the following string constants corresponding to numeric, text, and date formatting, or whether to skip the column when importing data: AutoFormat, Text, MDY, DMY, YMD, MYD, DYM, YDM, or Skip.

If this element is not specified, the default formatting value is AutoFormat.
