Data Element (Microsoft Excel)

Contains a caption text or a reference to the cell containing it, or contains a description of the records in a PivotTable cache.

Contained In

BubbleSize, Caption, Category, Close, High, Low, Open, PivotCache, Value, XValue, YValue

Subelements

For a PivotTable cache: Row

Remarks

A chart title caption string can be formatted as rich text. A cell reference is a text string that includes an equal sign (=) immediately followed by an A1-style cell reference.

For a PivotTable cache, all subelements are required.
