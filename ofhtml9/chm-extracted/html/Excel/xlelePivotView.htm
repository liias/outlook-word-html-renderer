PivotView Element (Microsoft Excel)

Contains the view properties of a PivotList component.

Contained In

PivotTable

Subelements

CellsExpanded, CellsExpandedSeqNum, CellsNotExpanded, DetailMaxHeight, DetailMaxWidth, DetailRowHeight, DetailSortOrder, DetailWidth, FieldLabelFormat, IsNotFiltered, Label, MemberFormat, NoDetailAutoFit, TotalAllMembers, TotalFormat

Remarks

All subelements are optional. Either the CellsExpanded or CellsNotExpanded element can be specified, but not both at the same time. More than one Label subelement can be specified.
