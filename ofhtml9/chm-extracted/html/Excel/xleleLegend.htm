Legend Element (Microsoft Excel)

Contains the properties of a legend.

Contained In

Chart, WebChart

Subelements

Border, Font, Height, Interior, Left, LegendEntry, Placement, Top, Type, Width

Remarks

All subelements are optional. The Placement subelement contains the string constant None if a legend is hidden.

If this element is not specified, the default value of each subelement is Automatic, specifying the default formatting, positioning, and style.
