NoDetailAutoFit Element (Microsoft Excel)

Specifies whether the height and width of the detail grid are determined by the height and width of the detail fields.

Contained In

PivotView

Remarks

If <x:NoDetailAutoFit/> is specified, the height and width of the detail grid are specified by the DetailWidth element. If the NoDetailAutoFit element is not specified, the DetailMaxHeight and DetailMaxWidth elements specify the maximum height and width that the detail grid can attain.
