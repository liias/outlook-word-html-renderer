RefreshInfo Element (Microsoft Excel)

Contains information about data refresh.

Contained In

QueryTable

Subelements

AlertVersion, CaseSensitive, ColumnInfo, DeletedTitle, DoNotPersist, DoNotPersistSort, DoNotPersistAF, ExtraLeftColumns, ExtraRightColumns, FuturePersist, HeaderRow, IDWrapped, NextID, NoTitles, Sort, SortOrder

Remarks

The ColumnInfo, DeletedTitle, NextID, and SortOrder subelements are required, but all other subelements are optional. There can be more than one DeletedTitle subelement in a RefreshInfo element.
