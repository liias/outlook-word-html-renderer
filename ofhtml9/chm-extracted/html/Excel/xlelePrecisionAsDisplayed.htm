PrecisionAsDisplayed Element (Microsoft Excel)

Specifies whether calculations are performed using only the precision of numbers as they are displayed.

Contained In

ExcelWorkbook

Remarks

If <x:PrecisionAsDisplayed/> is specified, the calculations are performed using the precision of numbers as they are displayed. If this element is not specified, full precision is used.
