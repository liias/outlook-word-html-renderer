WaterLevel Element (Microsoft Excel)

Specifies the y position of a column, line, area, or surface chart, the x position of a bar chart, or the x-coordinate of the origin that data points are drawn from.

Contained In

Graph

Remarks

When the chart is published in Microsoft Excel, the WaterLevel element contains the x or y value where the category axis crosses the value axis.

When a chart is saved from the Chart component, the water level can be any value. If this element is not specified, the default value is 0 (zero).

This element can be specified for area, bar, column, line (including smooth), and surface charts.
