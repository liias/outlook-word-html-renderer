Rotation Element (Microsoft Excel)

Specifies the rotation angle, in degrees, of an object around the value (y) axis in a chart, or the rotation angle of text.

Contained In

Alignment, View3D

Remarks

If this element contains a negative angle, the object is rotated counter-clockwise, but if it contains a positive angle, the object is rotated clockwise. If this element contains 0 (zero), the object is not rotated.

For text, zero specifies no rotation. Positive degrees represent counter-clockwise rotation, and negative degrees represent clockwise rotation. If the Rotation element contains the string constant Vertical, the text appears vertically from top to bottom with no rotation.

For axis tick labels, this element can contain the string constant Automatic, specifying that Microsoft Excel automatically determines the direction of the text labels depending on the orientation of an axis and the space available for displaying the label.
