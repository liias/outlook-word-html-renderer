Crn Element (Microsoft Excel)

Contains a description of formula operands located in another workbook.

Contained In

Xct

Subelements

Boolean, ColFirst, ColLast, Error, Number, Row, Text

Remarks

The ColFirst, ColLast, Number, Row, and Text subelements are required, but all other subelements are optional.

If the operand references more than one row, there is one Crn element specified for each row. For example, if the array formula =EXT.XLS!A1:A4*4 is entered on a worksheet, there will be four Crn elements in an Xct element.

If a worksheet contains two different formulas and each formula has multiple operands that reference another workbook, one or more Crn elements can be specified depending on where the referenced cells are located. For example, suppose that a worksheet contains the following two formulas in different cells: =EXT.XLS!A1*2 and =EXT.XLS!B1*2. Because the operands are in the same row and not separated by an empty cell, one Crn element can be specified containing information about cells A1 and B1. However, if the operands are in different rows and separated by an empty cell, two Crn elements are specified.
