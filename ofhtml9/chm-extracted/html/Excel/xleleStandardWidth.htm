StandardWidth Element (Microsoft Excel)

Contains an integer value specifying the default width of columns in a worksheet.

Contained In

WorksheetOptions

Remarks

One unit of column width is equal to the width of one character in the Normal cell style of Microsoft Excel. For proportional fonts, the width of the character 0 (zero) is used. If this element is not specified, the default width is 8 units.
