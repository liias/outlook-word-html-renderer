Width Element (Microsoft Excel)

Specifies the width of a chart or an object within a chart area or Chart component, or the width of a PivotList component or a total in the component.

Contained In

Chart, Legend, PivotTable, PlotArea, PLTotal, Title, WebChart

Remarks

For the PivotList component, the width is an integer in pixels. The Height and Width elements are used when the NoAutoFit element is specified.

For the WebChart element, the Width element specifies the width of the chart in points. For all other objects, the Width element contains a number specifying the width as a fraction of the chart frame. If this element is not specified for the Chart component, the object is displayed at the default width.
