ParentField Element (Microsoft Excel)

Contains the name of a parent field. For a PivotList component, specifies the largest grouping field of an automatic date field set.

Contained In

PivotField, PLTotal

Remarks

The name is a text string.

For a PivotList component, the ParentField element can be specified in a PivotField element that represents the smallest grouping field in an automatic date field set. If the field represents a total, the ParentField element contains the name of the field being totaled.

For a PivotTable report on a worksheet, the ParentField element is valid only when the PivotField element describes a data field or grouped field.

For OLAP-based PivotTable reports, the name is the unique field name contained in the SourceName subelement of the PivotField element describing the parent field.

For all other data sources, the name is the displayed name contained in the Name subelement of the PivotField element describing the parent field.
