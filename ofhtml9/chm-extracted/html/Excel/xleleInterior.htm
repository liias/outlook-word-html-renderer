Interior Element (Microsoft Excel)

Contains the style and formatting of the interior of an object.

Contained In

Chart, DataLabels, DataPoint, DownBars, Floor, Graph, LabelEntry, Legend, LegendKey, Marker, PlotArea, Series, Title, UpBars, Walls, WebChart

Subelements

BGColorIndex, Color, ColorIndex, FillEffect, Invert, Shadow

Remarks

All subelements are optional. If this element is not specified, the default value of each subelement is Automatic, specifying the default formatting and style.

Example

See the example in the WebChart element topic.
