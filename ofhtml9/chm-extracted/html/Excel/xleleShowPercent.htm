ShowPercent Element (Microsoft Excel)

Specifies whether the percent contribution of a category to the entire series is included in a data label.

Contained In

DataLabels, LabelEntry

Subelements

Font, Number

Remarks

All subelements are optional.

This element can be specified for any type of stacked or 100% stacked series chart, and for pie and doughnut charts. If this element is specified, the percentage is shown.
