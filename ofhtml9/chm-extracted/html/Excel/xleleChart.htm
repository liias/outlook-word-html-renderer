Chart Element (Microsoft Excel)

Contains a description of a chart in a Chart component.

Contained In

WebChart

Subelements

Border, ChartGrowth, Floor, Font, Height, Interior, Left, Legend, Name, Options, PageSetup, PivotTableRef, PlotArea, RoundedCorners, Title, Top, View3D, Walls, Width

Remarks

The Name subelement is required, but all other subelements are optional.

A Web page can contain one or more charts. However, when the Web page is opened in Microsoft Excel, only the first chart is opened and all subsequent charts are ignored.
