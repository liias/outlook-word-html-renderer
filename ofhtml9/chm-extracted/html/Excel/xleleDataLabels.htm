DataLabels Element (Microsoft Excel)

Specifies the type and formatting of labels displayed next to the data points in a series.

Contained In

Graph, Series

Subelements

Alignment, Border, Delete, Font, Interior, LabelEntry, LabelSeparator, LeaderLines, Number, Position, Series, ShowBubbleSize, ShowCategoryName, ShowLeaderLines, ShowLegendKey, ShowPercent, ShowSeriesName, ShowValue

Remarks

All subelements are optional. If more than one Series subelement is specified, each subelement corresponds to a series in the order specified.

The ShowCategoryName, ShowLegendKey, ShowSeriesName, and ShowValue subelements can be specified for all types of series.
