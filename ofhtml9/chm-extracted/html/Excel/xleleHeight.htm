Height Element (Microsoft Excel)

Specifies the height of a chart, an object within a chart area or Chart component, or a PivotList component.

Contained In

Chart, Legend, PivotTable, PlotArea, Title, WebChart

Remarks

For the PivotList component, the height is an integer in pixels. The Height and Width elements are used when the NoAutoFit element is specified.

For the WebChart element, the Height element specifies the height of the chart in points. For all other objects, the Height element contains a number specifying the height as a fraction of the chart frame. If this element is not specified for the Chart component, the object is displayed at the default height.
