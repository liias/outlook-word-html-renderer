Weight Element (Microsoft Excel)

Specifies the thickness of a border or line.

Contained In

Border, Line

Remarks

This element contains one of the following string constants specifying the thickness: Hairline, Narrow, Medium, or Wide.

For the Chart component, the Weight element can contain an integer from 0 to 99. However, the value cannot be imported when the Web page is opened in Microsoft Excel.
