NoAllowGrouping Element (Microsoft Excel)

Specifies whether the user is able to group column or row fields on a PivotList component.

Contained In

PivotTable

Remarks

If <x:NoAllowGrouping/> is specified, pivoting is disabled. This means that membership in the row and column areas is fixed; you cannot drag a field to or from these areas. All commands related to grouping are disabled, including the following commands:

  • Move to Row Area
  • Move to Column Area
  • Move to Detail, if the selection is a field in the row or column area. The report layout is frozen, but the user can still filter the data.
The grouping buttons are disabled but the user can still group fields programmatically.
