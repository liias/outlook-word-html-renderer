Plus Element (Microsoft Excel)

Specifies the positive error bar value.

Contained In

ErrorBars

Subelements

Repeat

Remarks

This element contains numeric error values, a range of cells from the source data (if the data source is a cell-table), or a column label (if the data source is a database query), specifying the negative error bar value.

The Repeat subelement is specified if the value is repeated for all data points.
