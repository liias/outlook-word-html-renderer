HTMLTables Element (Microsoft Excel)

Specifies the HTML tables on a Web page that are imported into a query table.

Contained In

QuerySource

Subelements

Number, Text

Remarks

If this element is specified, only the specified HTML tables are imported. At least one of the subelements must be specified. The subelements can be specified more than once, and in any order.

If <x:EntirePage/> is specified, the entire Web page is imported. <x:EntirePage/> and the HTMLTables element must not be specified for a query table at the same time. If neither <x:EntirePage/> nor the HTMLTables element is specified, all HTML tables in the Web page are imported.
