DivID Element (Microsoft Excel)

Contains the unique identifier used for identifying an HTML Div element on a Web page. The element is associated with an item in a document that is published to a Web page. The item can be an entire workbook, a worksheet, a selected print range, an AutoFilter range, a range of cells, a chart, a PivotTable report, or a query table.

Contained In

PublishObject

Remarks

This element contains a text string.
