ElementType Element (Microsoft Excel)

Contains a description of a row schema.

Attribute Description
Name Required. The unique name of the row schema description.
Content Required. Contains the string constant EltOnly if only attribute subelements are allowed (and the ElementType and AttributeType subelements are not allowed).

Contained In

Schema

Subelements

Attribute, Extends

Remarks

All subelements are required.
