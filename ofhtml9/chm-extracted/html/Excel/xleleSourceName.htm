SourceName Element (Microsoft Excel)

Contains the name of a field or item as it appears in a PivotTable cache.

Contained In

PivotField, PivotItem, PLPivotField

Remarks

This element contains a string. The name is the recordset field name, or for OLAP data sources, the unique level or measure name. When the data comes from a PivotList component and the field is a page or filter field, the name is the hierarchy name. If there is no hierarchy name, it is the unique dimension name.

This element is specified if the name differs from the field or item name contained in the Name element.
