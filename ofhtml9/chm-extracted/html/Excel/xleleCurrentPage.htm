CurrentPage Element (Microsoft Excel)

Specifies the currently displayed filter value or page field item.

Contained In

PivotField, PLPivotField

Remarks

For a PivotList component based on an OLAP data source, this element contains the string filter value or the unique name of the filter member. If the CurrentPage element is not used and page items exist, the default string value is All, specifying all page items.

For a PivotTable report on a worksheet, the CurrentPage element contains a text string item name or the string constant All. This element can be specified only when the PivotField element describes a page field. If the CurrentPage element is not specified, the default page setting is All, which means all page field items are displayed.

For OLAP-based PivotTable reports, the name is the unique item name contained in the SourceName subelement of the PivotItem element describing the referenced item.

For all other data sources, the same name is contained in both the CurrentPage element and the Name subelement of the PivotItem element describing the referenced item.
