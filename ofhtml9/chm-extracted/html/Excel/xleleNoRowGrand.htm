NoRowGrand Element (Microsoft Excel)

Specifies whether a PivotTable report shows totals for rows.

Contained In

PivotTable

Remarks

If <x:NoRowGrand/> is specified, grand totals for rows are not shown.

For a PivotList component, the NoRowGrand element specifies whether the outermost column field is a subtotal. The NoColumnGrand element is specified for column fields. When a row or column field is not the outermost field, the Subtotal element is specified for the next field on the axis. When a field is not on the row or column axis, the PLSubtotal element specifies whether subtotals are displayed.
