Symbol Element (Microsoft Excel)

Specifies the appearance or shape of data markers.

Contained In

Marker

Remarks

This element contains one of the following string constants specifying the shape: Circle, Diamond, Dash, Dot, Plus, Square, Star, Triangle, or X.

If the constant None is specified, no markers are displayed. If the constant Automatic is specified, the Chart component or Microsoft Excel automatically determines the shape to use.
