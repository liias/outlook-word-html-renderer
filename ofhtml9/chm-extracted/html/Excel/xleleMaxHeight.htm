MaxHeight Element (Microsoft Excel)

Specifies the maximum height (in pixels) that a PivotList component can attain when it is adjusted.

Contained In

PivotTable

Remarks

This element contains an integer and is specified when the NoAutoFit element is not specified. A vertical scroll bar is displayed if there is more data available that can be displayed within the maximum height. The default maximum height is 32,000 pixels. The MaxWidth element specifies the maximum attainable width.
