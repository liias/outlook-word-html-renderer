DBSeries Element (Microsoft Excel)

Contains a description of database data that is used in a chart. This element allows you to specify which columns of data returned from a database query are used in a chart. The chart is drawn after a connection to the data source is made and data from a query is returned.

Contained In

DBSeries, Graph

Subelements

BubbleColumn, CategoryColumn, DBSeries, SeriesColumn, ValueColumn, XValueColumn, YValueColumn

Remarks

The DBSeries element is used only with the Chart component.

Depending on the type of chart, various subelements are required. The BubbleSize, XValueColumn, and YValueColumn subelements are specified for bubble, xy (scatter), surface, and other charts that use bubble size or category and value axes. For all other charts, the ValueColumn subelement is used.

One or more DBSeries subelements can be specified for bubble pie charts in which each bubble has its own series making up the pie. Each DBSeries subelement corresponds to a pie chart of a bubble.
