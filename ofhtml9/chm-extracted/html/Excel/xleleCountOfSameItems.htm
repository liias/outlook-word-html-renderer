CountOfSameItems Element (Microsoft Excel)

Contains the number of subsequent items that have the same properties as the current item.

Contained In

PTLineItem

Remarks

The number is a positive integer. If this element is specified, you need not specify the PTLineItem element for subsequent items that have the same properties as the current item. If the CountOfSameItems element is not specified, the default number is 0.
