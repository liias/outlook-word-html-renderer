Maximum Element (Microsoft Excel)

Specifies the maximum value of an axis scale or scale split.

Contained In

Scaling, SplitScale

Remarks

The value contained in this element must be greater than the value contained in the Minimum element.

For an axis scale, if the Maximum or Minimum element is not specified, the maximum or minimum value is automatically calculated from the data being charted. For dates and times, the scale uses integer values for days since January 1, 1900 or January 1, 1904, depending on whether the 1900 or 1904 date system is used. See the 1904Date element topic to determine the date system in use.

For a split scale, both Maximum and Minimum elements must be specified to define the upper and lower bounds of the omitted range of scaling.
