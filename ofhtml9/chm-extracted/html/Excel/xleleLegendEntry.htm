LegendEntry Element (Microsoft Excel)

Contains the custom formatting properties of an entry in a legend.

Contained In

Legend

Subelements

Delete, Font, Index, LegendKey

Remarks

The Index subelement is required, but all other subelements are optional.

For surface charts, the LegendEntry element specifies the custom formatting, if any, for an entry. You cannot specify the numeric range on the chart to be formatted. For example, the formatting for the first range on a surface chart would be within a LegendEntry element with an Index subelement containing the number 1.
