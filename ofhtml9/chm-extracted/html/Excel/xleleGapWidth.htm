GapWidth Element (Microsoft Excel)

Specifies the amount of spacing between markers in adjacent categories of a chart, or the width of up-down bars in a stock chart.

Contained In

Graph, UpDownBars

Remarks

This element contains a positive number. A value of zero specifies no space between adjacent category markers.

For bar of pie and pie of pie charts, the number contained in this element represents the space between the primary pie chart and the bar or secondary pie chart. If this element is not specified, the default space is 100. For other types of charts, the default space is 150.

For up-down bars, a width of 0 (zero) causes the bars to touch at category boundaries, while positive values increase the space between adjacent categories.
