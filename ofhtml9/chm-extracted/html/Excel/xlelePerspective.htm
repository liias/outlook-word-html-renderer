Perspective Element (Microsoft Excel)

Specifies the amount of visual depth shown in a chart.

Contained In

View3D

Remarks

This element contains a positive number. Lower values result in less depth, while higher values result in more depth and distortion. If 0 (zero) is specified, no perspective is displayed.

This element cannot be specified for charts that have been scaled to be closer in size and appearance to an equivalent 2-D chart.
