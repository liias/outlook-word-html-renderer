PTFormat Element (Microsoft Excel)

Contains structured formatting properties of a region of a PivotTable report.

Attribute Description
Style A text string containing subattributes that specify the font weight, style, and effects that are applied to a region of the PivotTable report.

Contained In

PivotTable

Subelements

FormatType, IndividualCellBorders, PTRule

Remarks

All subelements are optional. There can be one or more PTRule subelements.
