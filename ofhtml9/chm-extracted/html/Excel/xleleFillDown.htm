FillDown Element (Microsoft Excel)

Specifies whether formulas to the right of a query table that use values in this column are copied when the data is refreshed and new rows of data are added to a query table.

Contained In

ColumnInfo

Remarks

If <x:FillDown/> is specified, any formulas to the right of the query table that use values from this column are filled down. If fewer rows are returned during the refresh, any unused formulas are removed from the worksheet. If this element is not specified, the formulas are not filled down.
