DisableDrillDown Element (Microsoft Excel)

Specifies whether to allow the "drilled" (expanded, or visible) status of the cube field members in the hierarchical member-selection control of a cube field, whether double-clicking in the PivotTable report data area displays supporting records for the aggregate, and whether double-clicking a PivotTable item expands or collapses detail information.

Contained In

PivotTable

Remarks

If <x:DisableDrillDown/> is specified, the expanded state is not allowed.
