Position Element (Microsoft Excel)

Contains the index position of a PivotTable field within its current orientation, or specifies the position of a PivotTable field on an axis, the position of a chart title or data label, or the base position of a label entry.

Contained In

DataLabels, LabelEntry, PivotField, Title

Remarks

For a PivotTable field, this element contains an integer index. If this element is not specified, the default index is 1.

For a PivotTable field on a PivotList component, this element contains one of the following string constants specifying the field position on the axis: Column, Data, Page, or Row. If this element is not specified, the field is not on an axis.

For a chart title, this element is used only with the Chart component, and only when the Title element is a subelement of the Chart or WebChart element. The Position element contains one of the following string constants specifying the position of the title in relation to the chart: Automatic, Bottom, Left, Right, or Top. If this element is omitted, the default position is Automatic and the Chart component determines the position of the title.

For a label entry, this element specifies the base position of the entry. The DX and DY elements can be used to specify an offset to this position. The Position element contains one of the following string constants depending on chart type.
Chart Type Constant
Column or bar Center, InsideBase, InsideEnd, InsideLeft, InsideRight, OutsideBase (Chart component only), OutsideEnd, OutsideLeft, or OutsideRight
Pie BestFit, Center, InsideEnd, or OutsideEnd
Other Center, OutsideBottom, OutsideLeft, OutsideRight, or OutsideTop

If the Position element is not specified, the default position is one of the following string constants depending on the chart type.
Chart Type Constant
Column or bar OutsideEnd
Pie BestFit
Other OutsideRight

If the constant OutsideBase is specified and the Web page is opened in Microsoft Excel, the default value OutsideEnd is used instead.
