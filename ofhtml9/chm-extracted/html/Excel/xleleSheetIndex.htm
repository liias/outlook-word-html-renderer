SheetIndex Element (Microsoft Excel)

Contains the index number for a worksheet to which a name or cell reference applies.

Contained In

ExcelName, Xct

Remarks

For a name, the SheetIndex element is used when a workbook contains the same name in different worksheets. This element qualifies the name and associates it with a particular worksheet.

For the Xct element, the number is the index of a SheetName element containing the name of the worksheet.
