AutoShowRange Element (Microsoft Excel)

Specifies the order in which a range is displayed when a filter is applied to a PivotTable field.

Contained In

PivotField

Remarks

This element contains one of the following string constants specifying the order: Top for ascending order or Bottom for descending order.

If this element is not specified, the default order is Top.

Example

See the example in the PivotTable element topic.
