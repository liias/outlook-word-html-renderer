CubeField Element (Microsoft Excel)

Contains the properties of a cube field.

Contained In

PivotTable

Subelements

AllItemName, Caption, DefaultItem, Dimension, Measure, Orientation, UniqueName

Remarks

The Caption and UniqueName subelements are required, but all other subelements are optional. The Dimension subelement is required if the Measure subelement is not specified or if it contains the constant False.

This element and its subelements apply to OLAP-based PivotTable reports only. The CubeField element is also specified for such a report when the data comes from a PivotList component that uses an OLAP provider.
