Vertical Element (Microsoft Excel)

Specifies the vertical justification for text.

Contained In

Alignment

Remarks

This element contains one of the following string constants specifying the justification: Automatic, Bottom, Center, Distributed, Justify, or Top.

If Automatic is specified, Microsoft Excel automatically determines the vertical justification. Distributed can be specified only in East Asian language versions of Excel.
