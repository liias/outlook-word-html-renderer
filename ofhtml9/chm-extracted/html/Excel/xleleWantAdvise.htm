WantAdvise Element (Microsoft Excel)

Specifies whether the ExternName element contains a description of a DDE or OLE link, and specifies the type of link.

Contained In

ExternName

Remarks

If <x:WantAdvise/> is specified, the DDE or OLE link is automatic, but if this element is not specified, the link is manual.
