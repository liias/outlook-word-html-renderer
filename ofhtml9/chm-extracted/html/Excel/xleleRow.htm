Row Element (Microsoft Excel)

Specifies the values of columns in a record of a PivotTable cache, the row number of data in another workbook, or the row index of a row page break.

Attribute Description
ColumnName For a PivotTable cache, contains the data corresponding to a column of the source database table. If the attribute is not specified, the column is null.

Contained In

Crn, Data, RowBreak

Remarks

For a PivotTable cache, if a column is not aliased, the attribute name is the field name. However, if the column is aliased, the attribute name is the alias name.

Examples
