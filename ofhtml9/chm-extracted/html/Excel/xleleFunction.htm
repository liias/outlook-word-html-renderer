Function Element (Microsoft Excel)

Contains the function used to summarize a data field or total.

Contained In

PivotField

Remarks

This element can be specified only for data fields and totals.

For a PivotTable report on a worksheet, the Function element contains one of the following string constants specifying the function used: Avg, Count, CountNums, Max, Min, Product, StdDev, StdDevP, Sum, Var, or VarP. If this element is not specified, the default function is Sum.

For a PivotList component, the Function element contains one of the following string constants specifying the function used: Count, Max, Min, or Sum. If this element is not specified for a data field, a function is not used. If this element is not specified for a total, the default function of the total is Sum.

The Function element is not specified for totals bound to an OLAP data source.
