SubtotalFormat Element (Microsoft Excel)

Specifies the text formatting that is applied to a subtotal.

Attribute Description
Style A text string containing subattributes that describe the font weight, style, and effects that are applied to a subtotal.

Contained In

PivotField, PLPivotField

Remarks

For column and row fields, this element is not specified and the PTFormat element of the PivotTable element specifies the formatting.
