Parameter Element (Microsoft Excel)

Contains information about a query parameter.

Contained In

QuerySource

Subelements

ExcelType, Formula, Name, NonDefaultName, ParameterType, ParameterValue, PromptString, RefreshOnChange, SQLType

Remarks

The Name subelements is required. The ParameterValue subelement is required if the ParameterType subelement contains the string constant Value. The Formula subelement is required if the ParameterType subelement contains the string constant Formula. All other subelements are optional.
