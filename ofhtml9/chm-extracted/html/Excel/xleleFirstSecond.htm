FirstSecond Element (Microsoft Excel)

Specifies whether a point appears on the primary pie chart or on the pie or bar chart next to it.

Contained In

CustomSplit

Remarks

This element contains a string of numbers with no spaces. Each number corresponds to a data point, and the numbers appear sequentially starting with the first point. The number 1 specifies that the point appears on the primary pie chart, and the number 2 specifies that the point appears on the bar or secondary pie chart.
