DetailMaxWidth Element (Microsoft Excel)

Specifies the maximum width (in pixels) that the detail grid can attain when the size of the cell is automatically adjusted to fit the data.

Contained In

PivotView

Remarks

This element contains an integer. A scroll bar is displayed if the width of the detail grid would exceed the maximum width. The default width is 1024 pixels. If the NoDetailAutoFit element is specified, the DetailMaxHeight and DetailMaxWidth elements are not specified.
