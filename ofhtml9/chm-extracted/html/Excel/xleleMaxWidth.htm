MaxWidth Element (Microsoft Excel)

Specifies the maximum width (in pixels) that a PivotList component can attain when it is adjusted.

Contained In

PivotTable

Remarks

This element contains an integer and is specified when the NoAutoFit element is not specified. A horizontal scroll bar is displayed if there is more data available that can be displayed within the maximum width. The default maximum width is 32,000 pixels. The MaxHeight element specifies the maximum attainable height.
