Font Element (Microsoft Excel)

Contains the formatting and properties of a font.

Contained In

Axis, Chart, DataLabels, DataTable, LabelEntry, Legend, LegendEntry, PlotArea, ShowBubbleSize, ShowCategoryName, ShowPercent, ShowSeriesName, ShowValue, Title, WebChart

Subelements

AutoScale, B, Background, Color, ColorIndex, DoubleUnderline, FontName, I, Outline, S, Shadow, Size, Sub, Sup, U

Remarks

All subelements are optional.

If this element is not specified, Microsoft Excel or the Chart component uses the default font formatting and style of the parent element.
