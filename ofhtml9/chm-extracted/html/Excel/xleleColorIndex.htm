ColorIndex Element (Microsoft Excel)

Specifies the color of an object in a Microsoft Excel chart.

Contained In

Border, Font, Interior, Line

Remarks

This element contains an integer index to the workbook color palette. It can also contain one of the following string constants specifying the default color or no color: Automatic or None. If this element is not specified, the default color is Automatic which means that Microsoft Excel automatically determines the color.

This element is used only in Microsoft Excel charts. For the Chart component, see the Color element topic.
