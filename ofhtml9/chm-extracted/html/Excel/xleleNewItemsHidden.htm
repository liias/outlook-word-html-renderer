NewItemsHidden Element (Microsoft Excel)

Specifies whether new items that appear after a PivotTable is refreshed are hidden. For a PivotList component, specifies whether the filter function of a field includes only items in the field defined by the PivotField element.

Contained In

PivotField, PLPivotField

Remarks

If <x:NewItemsHidden/> is specified, new items are hidden, and for a PivotList component, only items in the field are included. This element is specified only when a PivotList component is opened and saved from Microsoft Excel, and it applies to the first refresh after the Web page is opened in Excel.
