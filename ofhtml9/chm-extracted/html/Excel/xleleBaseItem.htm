BaseItem Element (Microsoft Excel)

Contains the name of an item in the base field used for a custom calculation.

Contained In

PivotField

Remarks

The name is a text string. This element can be specified only when the PivotField element describes a data field.

For OLAP-based PivotTable reports, the name is the unique item name contained in the SourceName subelement of the PivotItem element describing the referenced item.

For all other data sources, the same name is contained in both the BaseItem element and the Name subelement of the PivotItem element describing the referenced item.
