ItemType Element (Microsoft Excel)

Specifies the function used to summarize data or the type of data contained in a PivotTable item.

Contained In

PTLineItem

Remarks

This element contains one of the following string constants specifying the function used to summarize data: Avg, Count, CountNums, Max, Min, Product, StdDev, StdDevP, Sum, Var, or VarP.

Or one of the following string constants specifying whether the item is blank, data, the default subtotal, or a grand total: Blank, Data, Default, or Grand. If this element is not specified, the default type is Data.
