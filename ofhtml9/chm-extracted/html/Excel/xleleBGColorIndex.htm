BGColorIndex Element (Microsoft Excel)

Specifies the background color of an object in a Microsoft Excel chart.

Contained In

Interior

Remarks

This element contains an integer index into the workbook color palette. This element can also contain one of the following string constants specifying the default color or no color: Automatic, Neutral, or None. If this element is not specified, the default color is Automatic which means that Microsoft Excel automatically determines the color.

This element is used only with Microsoft Excel charts. For the Chart component, see the Color element topic.
