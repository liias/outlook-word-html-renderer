1904Date Element (Microsoft Excel)

Specifies the date system used in a workbook.

Contained In

ExcelWorkbook

Remarks

Dates are represented by the integer number of days since January 1, 1900 or January 1, 1904. If <x:1904Date/> is specified, the 1904 date system is used, but if this element is not specified, the 1900 date system is used.
