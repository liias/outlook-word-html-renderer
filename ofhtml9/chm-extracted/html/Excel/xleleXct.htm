Xct Element (Microsoft Excel)

Contains information about references to cells in a worksheet of another workbook and about the data in those cells.

Contained In

SupBook

Subelements

Count, Crn, Moper, SheetIndex

Remarks

The Count and SheetIndex subelements are required, but all other subelements are optional.

One or more instances of either the Crn or Moper subelements can be specified. If the formula operand references more than one row in another workbook, there is one Crn subelement specified for each row.
