Alignment Element (Microsoft Excel)

For the PivotList component, specifies the horizontal alignment of a total. For other objects, specifies the direction, justification, and rotation angle for text.

Contained In

Axis, DataLabels, LabelEntry, PLTotal, Title

Subelements

Horizontal, ReadingOrder, Rotation, Vertical

Remarks

For the PivotList component, the Alignment element contains one of the following string constants specifying the horizontal alignment: Center, Left, or Right. If this element is not used, the default alignment is Automatic, specifying the same alignment as the field being totaled.

For other objects, all subelements are optional. If this element is not used, the default value of each subelement is Automatic, specifying the default direction, justification, and rotation angle.
