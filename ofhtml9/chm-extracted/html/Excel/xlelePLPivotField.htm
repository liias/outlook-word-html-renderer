PLPivotField Element (Microsoft Excel)

Contains the properties of an automatic date field. The PivotField element is used to contain properties of a PivotTable field, data field, field set, measure, or total.

Contained In

PivotTable

Subelements

BoundField, Caption, CurrentPage, DataType, DetailFormat, DetailWidth, Expanded, FilterCaption, GroupedWidth, Name, NewItemsHidden, NumberFormat, PivotItem, PLDataOrientation, PLPosition, PLSubtotal, ShowAllItems, SourceName, Subtotal, SubtotalFormat, TotalWidth

Remarks

The Name subelement is required, and there can be more than one PivotItem subelement.
