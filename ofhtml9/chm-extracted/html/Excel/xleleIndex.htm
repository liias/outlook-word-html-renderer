Index Element (Microsoft Excel)

Contains the index of a data label, data point, legend entry, series, or trendline series.

Contained In

DataPoint, LabelEntry, LegendEntry, Series, Trendline

Remarks

For a series or trendline series, this element contains a unique integer identifying the series within the collection and allows the same series to appear on different charts.

This element is not specified or contains 0 (zero) for a data label of a trendline.
