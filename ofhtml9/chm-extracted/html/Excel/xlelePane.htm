Pane Element (Microsoft Excel)

Contains the properties of a pane in a worksheet window, or specifies a pane containing the structured selection of a PivotTable report.

Contained In

Panes, Selection

Subelements

For a pane in a worksheet window: ActiveCol, ActiveRow, Number, RangeSelection

Remarks

For a pane in a worksheet window, all subelements are required. When the data comes from a PivotList component, the ActiveRow and Number subelements are required.

For a PivotTable report, this element contains one of the following string constants specifying the pane: BottomRight, TopRight, BottomLeft, or TopLeft. If this element is not specified, the default pane is TopLeft.
