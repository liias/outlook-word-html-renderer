DisplayErrorString Element (Microsoft Excel)

Specifies whether a user-defined error string is displayed in cells that contain errors.

Contained In

PivotTable

Remarks

If <x:DisplayErrorString/> is specified, the user-defined error string is used, but if this element is not specified, the default error string is used.

The user-defined error string is contained in the ErrorString element.
