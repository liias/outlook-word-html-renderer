Range Element (Microsoft Excel)

Contains the range or a list of ranges that apply to a conditional text format, data validation range, or data source.

Contained In

ConditionalFormatting, DataTable, DataValidation, PivotTableRef

Remarks

For a PivotChart report or chart data table, this element contains a single, A1-style range reference to a cell table. For a PivotChart report, the cell table is in a PivotTable report.

For all other objects, this element contains a text string specifying one or more comma-delimited, A1-style range references.
