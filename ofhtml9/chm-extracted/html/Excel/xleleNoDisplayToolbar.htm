NoDisplayToolbar Element (Microsoft Excel)

Specifies whether the toolbar on a PivotList component is displayed.

Contained In

PivotTable

Remarks

If <x:NoDisplayToolbar/> is specified, the toolbar is hidden. Hiding the toolbar does not change the height of the PivotList component, but it does cause a layout realignment. The width remains unchanged unless the NoAutoFit element contains True and the PivotList component was sized wider than necessary to accomodate the toolbar.
