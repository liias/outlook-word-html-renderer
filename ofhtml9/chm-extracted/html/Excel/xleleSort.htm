Sort Element (Microsoft Excel)

Specifies a range of cells or the name of a range to sort, or whether a query table has been sorted.

Contained In

RefreshInfo, Sorting

Remarks

When sorting a range, the name is a text string. When you specify a range, you must use an A1-style range reference.

If <x:Sort/> is specified for a query table, the table is sorted.
