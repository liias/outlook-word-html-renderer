Column Element (Microsoft Excel)

Specifies the index of a row page break in a worksheet, or for a PivotList component, represents a column of cells.

Contained In

ColBreak, PivotData

Subelements

For a PivotList component: Cell, Left, Path

Remarks

For a PivotList component, the Column element is specified only when the cells of the column do not appear in their default state. The Path subelement is required, but all other subelements are optional. You can specify more than one Cell subelement.
