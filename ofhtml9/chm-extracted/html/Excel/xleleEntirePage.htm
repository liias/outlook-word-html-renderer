EntirePage Element (Microsoft Excel)

Specifies whether the entire page is imported during a Web query.

Contained In

QuerySource

Remarks

If <x:EntirePage/> is specified, the entire Web page is imported, but if this element is not specified, only HTML tables are imported. If the HTMLTables element is specified, only the specified HTML tables are imported. If neither element is specified, all HTML tables in the Web page are imported. <x:EntirePage/> and the HTMLTables element must not be specified for a query table at the same time.
