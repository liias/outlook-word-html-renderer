AttributeType Element (Microsoft Excel)

Contains the description of a column schema.

Attribute Description
name Required. Specifies the index number and the name or alias of a field in the form Coln, where n is the index number.
rs:name Optional. Specifies the actual name of the source of a field. If the rs:name attribute is not specified, the name specified in the name attribute is used as both the actual field name and the field's alias.

Contained In

DataType

Remarks
