Border Element (Microsoft Excel)

Contains the formatting and style properties of a border.

Contained In

Chart, DataLabels, DataPoint, DataTable, DownBars, Floor, Graph, LabelEntry, Legend, LegendKey, Marker, PlotArea, Series, Title, UpBars, Walls, WebChart

Subelements

Color, ColorIndex, LineStyle, Weight

Remarks

All subelements are optional. If this element is not specified, the default value of each subelement is Automatic, specifying the default border formatting and style.
