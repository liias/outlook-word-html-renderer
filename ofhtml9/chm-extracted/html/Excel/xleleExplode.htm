Explode Element (Microsoft Excel)

Specifies the amount of explosion from the center point for a pie or ring slice.

Contained In

DataPoint, Series

Remarks

This element contains a percentage. For example, a value of 33.3 means that the slice is pulled away from the pie chart's center by one-third of the diameter. If this element is not specified, the amount of explosion of the parent series is used. If this element is not specified for the parent series, then the amount of explosion is 0 (zero), meaning that the pie or ring is not exploded.
