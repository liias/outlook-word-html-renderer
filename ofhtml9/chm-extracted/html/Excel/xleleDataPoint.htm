DataPoint Element (Microsoft Excel)

Contains the formatting of data points within a series.

Contained In

Series

Subelements

Border, Bubble3D, Explode, Index, Interior, Line, Marker

Remarks

The Index subelement is required, but all other subelements are optional. If the Marker subelement is not specified, the Chart component or Microsoft Excel automatically determines the marker formatting.

Whether the Border and Line subelements can be specified depends on the type of chart described in the Series element topic.
