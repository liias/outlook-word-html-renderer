Formula Element (Microsoft Excel)

Contains the formula corresponding to a defined name, a range in another workbook, a query parameter, or a PivotTable field, formula, or item.

Contained In

ExcelName, ExternName, Parameter, PivotField, PivotItem, PTFormula

Remarks

The formula is a text string that includes an equal sign (=) immediately followed by a Microsoft Excel formula.
