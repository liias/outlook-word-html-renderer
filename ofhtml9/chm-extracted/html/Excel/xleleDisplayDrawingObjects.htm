DisplayDrawingObjects Element (Microsoft Excel)

Specifies whether drawing objects are displayed, placeholders are displayed in their place, or all drawing objects are hidden in a workbook.

Contained In

ExcelWorkbook

Remarks

This element contains one of the following string constants: DisplayShapes, Placeholders, or HideAll. The constants specify whether the objects are displayed, placeholders are displayed instead, or all objects are hidden. If this element is not specified, the default value is DisplayShapes.
