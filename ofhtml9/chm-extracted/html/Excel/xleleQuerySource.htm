QuerySource Element (Microsoft Excel)

Contains information about the data source of a query.

Contained In

PTSource, QueryTable

Subelements

For a PivotTable report: CommandText, CommandTextOriginal, CommandType, Connection, LocalConnection, Maintain, NoRefreshCube, Parameter, QueryType, UseLocalConnection, VersionLastEdit, VersionLastRefresh, VersionRefreshableMin

For a query table: 97Query, CommandText, CommandType, Connection, DoNotJoinDelimiters, DoNotPromptForFile, EntirePage, HTMLFormat, HTMLTables, Maintain, NoTextToColumns, Parameter, QueryType, RefreshTimeSpan, TextWizardSettings, URLString, UseSameSettings, VersionLastEdit, VersionLastRefresh, VersionRefreshableMin, WebPostString

Remarks

The URLString subelement is required if the query table is based on a Web page. For a PivotTable report, the CommandText subelement is required. All other subelements are optional.

Examples

See the examples in the PivotTable and QueryTable element topics.
