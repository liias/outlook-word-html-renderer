Item Element (Microsoft Excel)

Contains the name of a child item in a PivotTable field, or the index number of an item described by the PTLineItem element.

Contained In

MapChildItems, PTLineItem, Reference

Remarks

For a child item, this element contains a text string name.

For OLAP-based PivotTable reports, the name is the unique item name contained in the SourceName subelement of the PivotItem element describing the referenced child item.

For all other data sources, the name is the displayed name contained in the Name subelement of the PivotItem element describing the referenced child item.

If Item is a subelement of the PTLineItem element, the Item element contains an integer.
