DX Element (Microsoft Excel)

Contains an offset to the default placement of data labels.

Contained In

LabelEntry

Remarks

If this element is not specified, the labels are not offset.

For a pie chart, the DX element contains the offset angle in radians, and the DY element contains the radial offset.

For a bar or column chart, the DX element contains an offset perpendicular to the bar or column, and the DY element contains an offset parallel to the bar or column.

For all other types of charts, the DX and DY elements contain the horizontal and vertical offsets, respectively, from the default position.
