Schema Element (Microsoft Excel)

Contains a description of the fields in a PivotTable cache.

Attribute Description
ID Specifies the name of the schema.

Contained In

PivotCache

Subelements

AttributeType, ElementType

Remarks

All subelements are required. The ElementType subelement has a Name attribute containing the string constant Row.

Example

See the example in the PivotCache element topic.
