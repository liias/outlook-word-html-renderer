Label Element (Microsoft Excel)

Contains the properties of a PivotTable field or view label, and specifies the text formatting that is applied to the label.

Attribute Description
Style A text string containing subattributes that describe the font weight, style, and effects that are applied to the label when it is visible.

Contained In

PivotAxis, PivotView

Subelements

Caption, NotVisible

Remarks

All subelements are optional.
