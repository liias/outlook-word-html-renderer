Title Element (Microsoft Excel)

Specifies whether an object in a chart has a title, and contains the title and its formatting.

Contained In

Axis, Chart, WebChart

Subelements

Alignment, Border, Caption, Font, Height, Interior, Left, Position, Top, Width

Remarks

If this element is specified, the object has a title. The Caption subelement is required, but all other subelements are optional.
