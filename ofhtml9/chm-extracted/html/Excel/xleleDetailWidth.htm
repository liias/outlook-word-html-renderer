DetailWidth Element (Microsoft Excel)

Specifies the width of a field (in pixels) when it appears on the detail axis.

Contained In

PivotField, PivotView, PLPivotField

Remarks

This element is not used if the width of the field is automatically adjusted to accomodate the data. The NoDetailAutoFit element specifies whether the height and width of the detail grid is determined by the height and width of the detail fields.
