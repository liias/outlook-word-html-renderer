Trendline Element (Microsoft Excel)

Contains the definition of a trendline or regression line over a series.

Contained In

Series

Subelements

Backward, Caption, DisplayEq, DisplayRSqr, Forward, Index, Intercept, LabelEntry, Line, Name, Type, Value

Remarks

The Type subelement is required, but all other subelements are optional. The Value subelement is required for a polynomial or moving average trendline.
