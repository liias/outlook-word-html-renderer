XValue Element (Microsoft Excel)

Specifies the x values of data points in a chart that supports true 2-D drawing, such as a bubble or scatter chart.

Contained In

Series

Subelements

For the Chart component: Data, DataSource

Remarks

This element contains either a string or subelements depending on whether the chart was saved from Microsoft Excel or the Chart component. For Excel, this element contains an A1-style cell reference or a literal data string such as {1, 2, 3,...}, {"a", "b", "c",...}, or apples. For the Chart component, the Data and DataSource subelements are required.

The values or range is typically plotted along the category (x) axis, while the values or range contained in the YValue element are typically plotted along the value (y) axis.
