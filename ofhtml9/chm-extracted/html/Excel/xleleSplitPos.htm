SplitPos Element (Microsoft Excel)

Specifies the placement of points on a bar or secondary pie chart based on the string constant contained in the SplitType element.

Contained In

Graph

Remarks

This element contains a numeric value or percentage.

If SplitType contains SplitPos contains
Percent a percentage. Points having a size smaller than this percentage are placed on the bar or secondary pie chart.
Position an integer. Points having an index from 0 to this integer are placed on the bar or secondary pie chart.
Value a number. Points having a value less than this number are placed on the bar or secondary pie chart.
