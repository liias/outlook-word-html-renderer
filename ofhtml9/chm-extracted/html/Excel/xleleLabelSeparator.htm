LabelSeparator Element (Microsoft Excel)

Specifies the character separating each entry included within a data label.

Contained In

DataLabels, LabelEntry

Remarks

This element contains one of the following string constants: Comma, Newline, or Space. This element can also contain a custom separator. For example, to specify the semicolon as a separator: <x:LabelSeparator>;</x:LabelSeparator>.
