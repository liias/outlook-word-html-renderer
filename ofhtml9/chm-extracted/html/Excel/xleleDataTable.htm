DataTable Element (Microsoft Excel)

Specifies whether a data table is displayed on the plot area next to the category axis.

Contained In

PlotArea

Subelements

Border, ColumnInputCell, Font, Range, RowInputCell, ShowHorzBorder, ShowKeys, ShowOutline, ShowVertBorder

Remarks

This element should only be specified for charts that have a category axis. Data tables are not available in pie, xy (scatter), doughnut, bubble, radar, or surface charts.

All subelements are optional. If the DataTable element is not specified, the data table is not displayed.

This element is used only in Microsoft Excel.
