Connection Element (Microsoft Excel)

Contains the connection string of a data source, or the location of the data for a chart.

Contained In

DataSource, QuerySource

Remarks

This element is specified only for ODBC sources. The same connection string can be split among several consecutive Connection elements.

For a chart data source, the Type element specifies the type of data.

Examples

See the examples in the PivotTable, QueryTable, and WebChart element topics.
