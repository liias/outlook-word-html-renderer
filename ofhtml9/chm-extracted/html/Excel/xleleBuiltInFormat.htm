BuiltInFormat Element (Microsoft Excel)

Specifies the index of a built-in number format to apply.

Contained In

Number

Remarks

The index is an integer. For example, to use the General number format, specify: <Number><BuiltInFormat>0</BuiltInFormat></Number>.

For a list of built-in number formats, see the NumberFormat element topic. For a list of indices and format strings, see the Microsoft Excel 97 Software Developer's Kit.
