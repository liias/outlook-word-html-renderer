Caption Element (Microsoft Excel)

Contains the displayed name of a PivotTable field, the title of a chart, or the caption of a data label, series, Spreadsheet component, or trendline. For a PivotList component, contains the axis label, displayed name of a total, or the current filter value or page name.

Contained In

CubeField, Label, LabelEntry, PivotField, PLPivotField, PLTotal, Series, Trendline, Title

Subelements

For a chart: Data, DataSource

Remarks

For a chart title or label entry, all subelements are required. For all other objects, this element contains a text string caption. The DataSource subelement is required by the Chart component.

A chart title caption string can be formatted as rich text. If the Caption element is not specified, Microsoft Excel automatically generates a title caption using the string constant Chart and the chart index number as the suffix. Likewise, the default caption of a trendline is the string constant Trendline and the trendline index number suffix.

For a series in Microsoft Excel, the name contained in the Caption and Name elements are the same. In the Chart component, the names in these elements can be different.

For a PivotList component, the caption is contained in the Name and PLCaption elements. The Caption subelement is not specified if the name is the same as the content of the Name subelement of the same parent element.
