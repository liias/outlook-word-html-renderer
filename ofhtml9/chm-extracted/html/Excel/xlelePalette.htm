Palette Element (Microsoft Excel)

Specifies user-defined colors in the color palette for a chart.

Contained In

WebChart

Subelements

Entry

Remarks

At least one Entry subelement must be specified. Each subelement corresponds to one of 56 user-definable colors in a 64-color palette. The 8 basic colors cannot be changed. All charts within the WebChart element share the same palette of colors.
