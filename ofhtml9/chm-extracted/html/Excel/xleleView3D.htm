View3D Element (Microsoft Excel)

Contains the definition of a view for a 3-D chart, including the direction of the view, the amount of perspective, and how the chart is rendered.

Contained In

Chart

Subelements

AutoScale, ChartDepth, Elevation, HeightPercentage, Perspective, RightAngleAxes, Rotation, Walls2D

Remarks

This element is used only with Microsoft Excel charts. All subelements are optional. If no subelements are specified, Microsoft Excel automatically determines the default formatting for the view.
