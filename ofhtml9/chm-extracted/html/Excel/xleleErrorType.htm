ErrorType Element (Microsoft Excel)

Specifies the type of error bars.

Contained In

ErrorBars

Remarks

This element contains one of the following string constants specifying the type: Custom, FixedValue, Percentage, StdDev, or StdErr.

If the constant StdErr is specified, the Value element contains the error value. If the constant Custom is specified, the Plus and Minus elements must be specified, depending on the type of bar defined in the BarType element.
