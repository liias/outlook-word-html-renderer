ExcelWorksheets Element (Microsoft Excel)

Contains the ExcelWorksheet elements corresponding to the worksheets of a workbook.

Contained In

ExcelWorkbook

Subelements

ExcelWorksheet

Remarks

This element contains one or more ExcelWorksheet subelements. Only one subelement is specified when the data comes from a PivotList component.
