BaseField Element (Microsoft Excel)

Contains the name of a base field used for a custom calculation or, for a group field, describes the field the grouping information is based on. For a PivotList component, specifies the smallest grouping field of a field set of automatic dates.

Contained In

PivotField

Remarks

The name is a text string.

For a PivotList component, the BaseField element is not specified for the smallest grouping level.

For a PivotTable report on a worksheet, the BaseField element is valid only when the PivotField element describes a data field or grouped field.

For OLAP-based PivotTable reports, the name is the unique field name contained in the SourceName subelement of the PivotField element describing the referenced field.

For all other data sources, the same name is contained both in the BaseField element and in the Name subelement of the PivotField element describing the referenced field.
