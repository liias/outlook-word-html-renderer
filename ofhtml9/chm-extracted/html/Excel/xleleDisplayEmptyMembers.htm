DisplayEmptyMembers Element (Microsoft Excel)

Specifies whether members having no data are visible on an axis when the axis has no fields that contain data.

Contained In

PivotAxis

Subelements

Remarks

If <x:DisplayEmptyMembers/> is specified, the members having no data are displayed. If the axis contains fields that have data, the ShowAllItems subelement of the PivotField or PLPivotField element is specified and the DisplayEmptyMembers element is not specified.
