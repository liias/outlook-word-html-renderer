DataSourceControl Element (Microsoft Data Source Control)

Contains the properties of a Microsoft Data Source Control.

Contained In

XML

Subelements

ConnectionString, DataEntry, DataModel, ElementExtension, GridX, GridY, GroupLevel, MaxRecords, OWCVersion, UseRemoteProvider

Remarks

The ConnectionString, DataModel, ElementExtension, GridX, GridY, MaxRecords, and OWCVersion subelements are required. More than one ElementExtension and GroupLevel subelement can be specified. All other elements are optional.
