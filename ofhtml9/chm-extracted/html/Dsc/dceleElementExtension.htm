ElementExtension Element (Microsoft Data Source Control)

Contains additional properties of the Data Source Control component.

Contained In

DataSourceControl

Subelements

ChildLabel, ConsumesRecordset, ControlSource, ElementID

Remarks

The ElementID subelement is required, but all other subelements are optional. For controls bound to data from the Data Source Control, either the ConsumesRecordset or ControlSource subelement is specified, but not both.

When a combo box, check box, list box, or option button control is used as a restrictor control, the ControlSource subelement is not specified.
