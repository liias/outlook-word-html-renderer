Sublist Element (Microsoft Data Source Control)

Contains the properties of a sublist relationship.

Attribute Description
Relationship One of the following strings specifying the type of sublist relationship: ManySide or OneSide.
Sublist The identifier of the sublist relationship.

Contained In

RecordSetDef

Subelements

RelatedField

Remarks

A sublist relationship refers to a row source in another recordset definition that has a many-to-one relationship with the primary page row source in the specified recordset definition. All sources have the same recordset definition as the parent of a one-to-many relationship. Records in the child of a sublist relationship are retrieved only when they are needed.
