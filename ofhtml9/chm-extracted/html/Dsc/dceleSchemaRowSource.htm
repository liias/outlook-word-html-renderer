SchemaRowSource Element (Microsoft Data Source Control)

Contains the properties of a schema row source. Every table, view, or stored procedure in the database is a potential schema row source, as are commands defined explicitly for the Data Source Control component.

Attribute Description
ID A number identifying the row source.
Type One of the following strings specifying a page relationship or schema row source: dscPageRelTypeEnum or dscRowsourceTypeEnum.

Contained In

DataModel

Subelements

SchemaField

Remarks

All subelements are optional. More than one SchemaField subelement can be specified.
