PixelsPerInch Element (Microsoft Office)

Specifies the density (pixels per inch) of graphics and table cells on a Web page.

Contained In

OfficeDocumentSettings

Remarks

This element contains an integer. The range of settings is usually from 19 to 480, and common settings for popular screen sizes are 72, 96, and 120. If this element is not specified, the default density is 96.

The density determines the size of the images and cells on the specified Web page relative to the size of text whenever the saved document is viewed in a Web browser. The physical dimensions of the resulting image or cell are the result of the original dimensions (in inches) multiplied by the number of pixels per inch. The TargetScreenSize element specifies the optimal screen size for targeted Web browsers.
