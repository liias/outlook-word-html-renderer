RelyOnVML Element (Microsoft Office)

Specifies whether image files are generated from drawing objects and used on a Web page.

Contained In

OfficeDocumentSettings

Remarks

If <o:RelyOnVML/> is specified, Vector Markup Language (VML) is used instead of image files.

File sizes can be reduced by not generating images for drawing objects. However, to view the images, the targeted Web browsers must support VML otherwise the image will not be visible.
