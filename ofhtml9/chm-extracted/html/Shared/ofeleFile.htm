File Element (Microsoft Office)

Specifies the path and file name of a supporting file.

Attribute Description
HRef Required. Specifies the path and file name of a supporting file.
PublicationID Optional. Specifies an identifier for the file.

Contained In

XML

Remarks

The File element is empty. This element is contained in a file named filelist.xml.
