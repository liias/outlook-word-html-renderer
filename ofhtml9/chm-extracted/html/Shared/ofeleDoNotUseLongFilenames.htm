DoNotUseLongFilenames Element (Microsoft Office)

Specifies whether long file names are used in a Web page.

Contained In

OfficeDocumentSettings

Remarks

If <o:DoNotUseLongFilenames/> is specified, long file names are not used and the DOS file name format (8.3) is used.

If long file names are not used and the document has supporting files, those files are automatically stored in a separate folder. Otherwise, the DoNotOrganizeInFolder element determines whether supporting files are organized in a separate folder.
