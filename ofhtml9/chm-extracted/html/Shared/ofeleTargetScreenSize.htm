TargetScreenSize Element (Microsoft Office)

Specifies the ideal minimal screen size (width by height, in pixels) that should be used when viewing a Web page.

Contained In

OfficeDocumentSettings

Remarks

This element contains one of the following string constants specifying the screen size: 544x376, 640x480, 720x512, 800x600, 1024x768, 1152x882, 1152x900, 1280x1024, 1600x1200, 1800x1440, or 1920x1200. If this element is not specified, the default size is 800x600.
