DownloadComponents Element (Microsoft Office)

Specifies whether the necessary Microsoft Office Web components are downloaded when a page is opened in a Web browser and the components are not already installed.

Contained In

OfficeDocumentSettings

Remarks

If <o:DownloadComponents/> is specified, the components are downloaded if necessary.

The LocationOfComponents element specifies a central URL (on the intranet or Web) or path (local or network) to a location from which authorized users can download components when viewing the Web page. The path must be valid and point to a location that contains the necessary components, and the user must have a valid Microsoft Office 2000 license.

Office Web components add interactivity to Web pages. If a page is viewed in a Web browser on a computer that does not have the components installed, a graphic appears in place of the interactive portions of the page.
