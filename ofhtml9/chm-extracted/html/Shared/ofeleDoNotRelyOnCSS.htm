DoNotRelyOnCSS Element (Microsoft Office)

Specifies whether cascading style sheets (CSS) and HTML Font elements are used for font formatting when a page is opened in a Web browser.

Contained In

OfficeDocumentSettings

Remarks

If <o:DoNotRelyOnCSS/> is specified, CSS and Font elements are not used.

Cascading style sheets provide more precise layout and formatting control on a Web page and make the document look more as it appears in Microsoft Office. A cascading style sheet file is saved to either a new folder or the same folder as the Web page, depending on whether the DoNotOrganizeInFolder element is specified.
