DoNotOrganizeInFolder Element (Microsoft Office)

Specifies whether all supporting files, such as background textures and graphics, are organized in a separate folder.

Contained In

OfficeDocumentSettings

Remarks

If <o:DoNotOrganizeInFolder/> is specified, the supporting files are stored in the same folder as the Web page.

If this element is not specified, the files are stored in a separate folder in the same folder where the Web page has been saved, but it is named after the document. If long file names are used, a suffix is added to the folder name. If long file names are not used (that is, if the DoNotUseLongFilenames element is specified), any supporting files are automatically saved in a separate folder (not the same folder as the Web page).
