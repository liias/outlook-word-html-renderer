WPSpaceWidth Element (Microsoft Word)

Specifies whether Word sets the width of a space as it is in WordPerfect 5.x.

Contained In

WordCompatibility

Remarks

If <w:WPSpaceWidth/> is specified, the Set the width of a space like WordPerfect 5.x check box is selected in the Compatibility tab of the Options dialog box.
