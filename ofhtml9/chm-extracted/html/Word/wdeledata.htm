Data Element (Microsoft Word)

Specifies binary data in a form field.

Contained In

Xml

Remarks

This element contains a string value that is the binary data.

Example

This example shows the binary data stored in a FormCheckBox field.

<w:data>FFFFFFFF650000001400060043006800650063006B003100000000000000000000000000000000000000000000000000</w:data>
