WPJustification Element (Microsoft Word)

Specifies whether Word uses full text justification, as it is in WordPerfect 6.x for Windows.

Contained In

WordCompatibility

Remarks

If <w:WPJustification/> is specified, the Do full justification like WordPerfect 6.x for Windows check box is selected in the Compatibility tab of the Options dialog box.
