ClickAndTypeStyle Element (Microsoft Word)

Specifies a style that is automatically applied when using the Click and Type. 

Attribute Description
Number Specifies the heading level for built-in heading styles only. For example, if Heading 1 is the built-in style that is automatically applied when using Click and Type, this attribute would contain the value 1.

Contained In

WordDocument

Remarks

This element contains the name of the style that is automatically applied when using Click and Type. The default value is Normal, which indicates the Normal paragraph style.
