BorderTop Element (Microsoft Word)

Specifies whether there is a top border line for a picture or horizontal line object.

Attribute Description
shadow Specifies the effects for the border line.
type Specifies the type of border line.
width Specifies the width of the border line.

Remarks

If <w:bordertop/> is specified, the picture or horizontal line object has a top border. This element also specifies the display characteristics of the top border line.

See Also

BorderBottom, BorderLeft, BorderRight

Example

This example shows an inserted picture with border lines that have a shadow effect on all sides.

 <w:borderleft type="single" width="8" shadow="t"/>
