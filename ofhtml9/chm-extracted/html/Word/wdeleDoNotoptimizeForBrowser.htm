DoNotOptimizeForBrowser Element (Microsoft Word)

Specifies whether the Word browser optimization features are disabled for the document.

Contained In

WordDocument

Remarks

If <w:DoNotOptimizeForBrowser/> is specified, the browser optimization features are disabled.

See Also

BrowserLevel

Example

This example shows a document with browser optimization features disabled.

 <w:WordDocument>
  <w:DoNotOptimizeForBrowser/>
 </w:WordDocument>
