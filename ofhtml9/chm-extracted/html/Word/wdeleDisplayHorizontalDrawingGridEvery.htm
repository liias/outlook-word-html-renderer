DisplayHorizontalDrawingGridEvery Element (Microsoft Word)

Specifies that horizontal gridlines be displayed with a designated amount of space between them.

Contained In

WordDocument

Remarks

This element contains an integer value between 0 and 30 that determines the amount of space between horizontal gridlines. The default value is 1.
