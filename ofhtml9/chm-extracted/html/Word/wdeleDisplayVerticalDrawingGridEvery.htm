DisplayVerticalDrawingGridEvery Element (Microsoft Word)

Specifies that vertical gridlines be displayed with a designated amount of space between them.

Contained In

WordDocument

Remarks

This element contains an integer value between 0 and 30 that determines the amount of space between vertical gridlines. The default value is 1.

See Also

DisplayHorizontalDrawingGridEvery, DrawingGridHorizontalOrigin, DrawingGridHorizontalSpacing, DrawingGridVerticalOrigin, DrawingGridVerticalSpacing, UseMarginsForDrawingGridOrigin

Example

This example shows a document with drawing grid features enabled.

 <w:WordDocument>
  <w:View>Print</w:View>
  <w:DrawingGridHorizontalSpacing>14.4 pt</w:DrawingGridHorizontalSpacing>
  <w:DrawingGridVerticalSpacing>14.4 pt</w:DrawingGridVerticalSpacing>
  <w:DisplayHorizontalDrawingGridEvery>2</w:DisplayHorizontalDrawingGridEvery>
  <w:DisplayVerticalDrawingGridEvery>2</w:DisplayVerticalDrawingGridEvery>
  <w:UseMarginsForDrawingGridOrigin/>
  <w:DrawingGridHorizontalOrigin>144 pt</w:DrawingGridHorizontalOrigin>
  <w:DrawingGridVerticalOrigin>144 pt</w:DrawingGridVerticalOrigin>
 </w:WordDocument>
