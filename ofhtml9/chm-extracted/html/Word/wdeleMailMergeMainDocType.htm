MailMergeMainDocType Element (Microsoft Word)

Specifies a mail merge main document type.

Contained In

WordDocument

Remarks

This element contains one of the following values.

  • Catalog

  • Envelopes

  • FormLetters

  • MailingLabels

  • NotAMergeDocument (default)
