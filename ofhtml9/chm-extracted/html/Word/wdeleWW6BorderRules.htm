WW6BorderRules Element (Microsoft Word)

Specifies whether Word 2000 uses Word 6.x and 95 border rules.

Contained In

WordCompatibility

Remarks

If <w:WW6BorderRules/> is specified, the Use Word 6.x/95 border rules check box is selected in the Compatibility tab of the Options dialog box.
