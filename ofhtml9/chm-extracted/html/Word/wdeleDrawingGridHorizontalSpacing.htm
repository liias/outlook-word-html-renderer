DrawingGridHorizontalSpacing Element (Microsoft Word)

Specifies the amount of horizontal space between vertical gridlines.

Contained In

WordDocument

Remarks

This element contains a value that sets the horizontal distance between gridlines in points. If this element is not specified, the horizontal space between gridlines is 9.35 pt, the default value.
