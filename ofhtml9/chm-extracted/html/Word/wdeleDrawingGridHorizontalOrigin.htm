DrawingGridHorizontalOrigin Element (Microsoft Word)

Specifies that a drawing grid originates at a set distance from the left edge of a page.

Contained In

WordDocument

Remarks

This element contains a value that sets the drawing grid's distance from the left edge of the page, in points. If this element is not specified, the grid originates at the left margin, which is the default setting.
