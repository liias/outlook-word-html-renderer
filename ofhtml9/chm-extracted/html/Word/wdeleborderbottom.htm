BorderBottom Element (Microsoft Word)

Specifies whether there is a bottom border line for a picture or horizontal line object.

Attribute Description
shadow Specifies the effects for the border line.
type Specifies the type of border line.
width Specifies the width of the border line.

Remarks

If <w:borderbottom/> is specified, the picture or horizontal line object has a bottom border. This element also specifies the display characteristics of the bottom border line.
