DocumentKind Element (Microsoft Word)

Specifies a document type.

Contained In

WordDocument

Remarks

This element may contain one of the following values:

Value Description
DocumentEmail The document is an electronic mail
DocumentLetter The document is a letter
DocumentNotSpecified (default) The document is not an electronic mail or a letter

See Also

EnvelopeVis

Example

This is an example of an e-mail document.

 <w:WordDocument>
  <w:DocumentKind>DocumentEmail</w:DocumentKind>
  <w:EnvelopeVis/>
 </w:WordDocument>
