ConvMailMergeEsc Element (Microsoft Word)

Specifies whether Word treats the characters \" as if they are the characters "" in a mail merge data source.

Contained In

WordCompatibility

Remarks

If <w:ConvMailMergeEsc/> is specified, the Treat \" as "" in mail merge data sources check box is selected in the Compatibility tab of the Options dialog box.
