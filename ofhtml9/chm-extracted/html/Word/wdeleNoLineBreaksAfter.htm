NoLineBreaksAfter Element (Microsoft Word)

Contains the characters that cannot end a line for a specified East Asian typography character set.

Attribute Description
Lang Specifies the ISO language ID to which the settings apply.

Contained In

WordDocument

Remarks

This element contains the string that specifies characters that cannot end a line. It is used only when East Asian language support is installed and enabled.
