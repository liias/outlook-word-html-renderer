SaveSubsetFonts Element (Microsoft Word)

Specifies that Word only embeds the TrueType font styles actually used in a document.

Contained In

WordDocument

Remarks

If <w:SaveSubsetFonts/> is specified, only used TrueType font styles are embedded. This element requires that the EmbedTrueTypeFonts element be used.
