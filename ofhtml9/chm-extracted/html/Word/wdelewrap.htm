Wrap Element (Microsoft Word)

Specifies whether text wraps around an AutoShape.

Attribute Description
anchorx Specifies the horizontal anchor to which the object is aligned, such as "Margin".
anchory Specifies the vertical anchor to which the object is aligned, such as "Paragraph".
side Specifies the sides around which Word wraps text.
type Specifies the text wrapping style.

Remarks

If <w:wrap/> is specified, text wraps around the AutoShape. This element also specifies the page alignment and text wrapping characteristics of the AutoShape.

See Also

AnchorLock

Example

This example shows an AutoShape aligned horizontally to the left edge of the page and vertically 50 points below the margin. It has Through text wrapping style on the right side of the object only and its anchor is locked.

<v:shape id="_x0000_s1029" type="#_x0000_t13" style='position:absolute;
 margin-left:0;margin-top:26.95pt;width:279pt;height:135pt;z-index:-1;
 mso-wrap-edited:f;mso-wrap-distance-left:4pt;mso-wrap-distance-top:5pt;
 mso-wrap-distance-right:4pt;mso-wrap-distance-bottom:5pt;
 mso-position-horizontal:left;mso-position-horizontal-relative:page;
 mso-position-vertical:absolute;mso-position-vertical-relative:margin'
 wrapcoords="16084 0 16084 3840 -116 5280 -116 16320 8477 17280 16084 17280 16084 21600 16316 21600 21368 11520 21716 11400 21658 10800 16316 0 16084 0"
 o:allowoverlap="f">
 <w:wrap type="through" side="right" anchorx="page" anchory="margin"/>
 <w:anchorlock/>
</v:shape>
