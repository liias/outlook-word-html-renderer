ActiveWritingStyle Element (Microsoft Word)

Specifies the writing style and grammar checker information.

Attribute Description
DLLVersion Grammar checker DLL version number.
Lang ISO language identification.
NLCheck Grammar checker interface.
VendorID Grammar checker vendor identification number.

Contained In

WordDocument

Remarks

This element contains a numeric value representing the writing style specified for the document.
