PrintFractionalCharacterWidths Element (Microsoft Word)

Specifies whether Word prints fractional character widths.

Contained In

WordDocument

Remarks

If <w:PrintFractionalCharacterWidths/> is specified, Word prints fractional character widths. This setting is only available for the Macintosh® operating system.

Example

This example shows a document with the print fractional widths setting enabled.

 <w:WordDocument>
  <w:PrintFractionalCharacterWidths/>
 </w:WordDocument>
