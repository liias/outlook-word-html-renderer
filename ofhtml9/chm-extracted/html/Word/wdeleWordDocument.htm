WordDocument Element (Microsoft Word)

Specifies a document's properties and settings.

Contained In

XML

Subelements

ActiveWritingStyle, AttachedTemplate, AutoHyphenation, BrowserLevel, ClickAndTypeStyle, ConsecutiveHyphensLimit, DisplayHorizontalDrawingGridEvery, DisplayVerticalDrawingGridEvery, DocumentKind, DocumentProtection, DocumentVariables, DoNotHyphenateCaps, DoNotOptimizeForBrowser, DoNotPrintRevisions, DoNotShadeFormData, DoNotShowRevisions, DrawingGridHorizontalOrigin, DrawingGridHorizontalSpacing, DrawingGridVerticalOrigin, DrawingGridVerticalSpacing, EmbedTrueTypeFonts, EnvelopeVis, FormsDesign, HideGrammaticalErrors, HideSpellingErrors, HyphenationZone, MailMergeActiveRecord, MailMergeAddressFieldName, MailMergeAsAttachment, MailMergeCheckErrors, MailMergeConnectString, MailMergeDataSource, MailMergeDataType, MailMergeDestination, MailMergeDoNotSuppressBlankLines, MailMergeHeaderSource, MailMergeLinkToQuery, MailMergeMailSubject, MailMergeMainDocType, MailMergeQueryString, MailMergeViewMergedData, NoLineBreaksAfter, NoLineBreaksBefore, PrintFormsData, PrintFractionalCharacterWidths, PrintPostScriptOverText, PrintTwoOnOne, PunctuationKerning, SaveFormsData, SaveSubsetFonts, SummaryLength, TrackRevisions, UpdateStylesOnOpen, UseMarginsForDrawingGridOrigin, View, WordCompatibility, Zoom

Remarks

The WordDocument element is required. Certain subelements will not output if they represent the default settings or values.

See Also

DocumentProperties, OfficeDocumentSettings

Example

This example specifies a Word document with many document settings enabled. 

 <w:WordDocument>
  <w:View>Print</w:View>
  <w:Zoom>BestFit</w:Zoom>
  <w:PrintPostScriptOverText/>
  <w:EmbedTrueTypeFonts/>
  <w:SaveSubsetFonts/>
  <w:HideSpellingErrors/>
  <w:HideGrammaticalErrors/>
  <w:ActiveWritingStyle Lang="EN-US" VendorID="64" DLLVersion="131077" NLCheck="1">3</w:ActiveWritingStyle>
  <w:AttachedTemplate HRef="Professional Report.dot"></w:AttachedTemplate>
  <w:UpdateStylesOnOpen/>
  <w:TrackRevisions/>
  <w:DoNotShowRevisions/>
  <w:DoNotPrintRevisions/>
  <w:AutoHyphenation/>
  <w:ConsecutiveHyphensLimit>2</w:ConsecutiveHyphensLimit>
  <w:HyphenationZone>14</w:HyphenationZone>
  <w:DoNotHyphenateCaps/>
  <w:SummaryLength>10</w:SummaryLength>
  <w:ClickAndTypeStyle Number="1">Heading 1</w:ClickAndTypeStyle>
  <w:BrowserLevel>MicrosoftInternetExplorer4</w:BrowserLevel>
 </w:WordDocument>
