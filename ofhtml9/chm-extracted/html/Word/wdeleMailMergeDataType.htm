MailMergeDataType Element (Microsoft Word)

Specifies the type of mail merge data source.

Contained In

WordDocument

Remarks

This element contains a value that describes the type of data (such as an Excel spreadsheet or Access database) and the method of data access (for example, by means of ODBC or DDE).
