PunctuationKerning Element (Microsoft Word)

Specifies whether kerning applies to both Latin text and punctuation.

Contained In

WordDocument

Remarks

If <w:PunctuationKerning> is specified, kening is applied.

This element is used only when East Asian language support is installed and enabled.

See Also
