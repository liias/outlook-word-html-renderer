BrowserLevel Element (Microsoft Word)

This element describes which Web browser features a document supports.

Contained In

WordDocument

Remarks

This element may contain the values MicrosoftInternetExplorer5, which is the default value, or MicrosoftInternetExplorer4. If the browser optimization features are disabled for the document, the DoNotOptimizeForBrowser element is specified instead.
